# Treasury Management

Monitor your treasury positions and available liquidity across all corridors.

## View All Positions

<CodeGroup>

```bash cURL
curl https://sandbox.settla.io/v1/treasury/positions \
  -H "Authorization: Bearer $API_KEY"
```

```javascript Node.js
const positions = await fetch("https://sandbox.settla.io/v1/treasury/positions", {
  headers: { "Authorization": `Bearer ${API_KEY}` },
}).then(r => r.json());

for (const pos of positions.positions) {
  console.log(`${pos.currency} @ ${pos.location}: ${pos.available}`);
}
```

</CodeGroup>

## View a Specific Position

```bash
curl https://sandbox.settla.io/v1/treasury/positions/GBP/uk-clearing \
  -H "Authorization: Bearer $API_KEY"
```

## Liquidity Report

Get a comprehensive liquidity overview including alert positions:

```bash
curl https://sandbox.settla.io/v1/treasury/liquidity \
  -H "Authorization: Bearer $API_KEY"
```

The report includes:
- Total available balance per currency
- Locked (reserved) amounts
- Positions that are below alert thresholds
