# Changelog

## v1.0.0 -- Current

Initial public release of the Settla API.

### Features
- Cross-border transfers via stablecoin rails (GBP, USD, EUR to NGN, GHS, KES)
- Smart routing with cost/speed/liquidity/reliability scoring
- Crypto deposit sessions with on-chain monitoring
- Bank deposit sessions with virtual accounts
- Payment links for shareable crypto payments
- Real-time treasury positions and liquidity reports
- Webhook notifications with HMAC-SHA256 verification
- Ledger accounts with full transaction history
- Transaction verification and lookup
- Per-tenant rate limiting and idempotency

### Supported Chains
- Tron (TRC-20)
- Ethereum (ERC-20)
- Polygon (ERC-20)

### Supported Stablecoins
- USDT (Tether)
- USDC (Circle)
